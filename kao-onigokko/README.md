# 顔であそぶ！ミニゲーム（Kids Face Game）

カメラに映る顔の動きでキャラクターを動かして、赤い敵をよける子供向けミニゲームです。

- 顔を動かすとキャラが動く
- 敵は上/下から出現し、時間が経つほど難しくなる
- ときどき大きい敵が出る
- アイテム（5秒で消える）を拾うと効果が発動

## 動かし方（ローカル実行）

ブラウザのカメラ権限が必要です。`file://` 直開きでは動かない場合があるため、ローカルサーバ経由で開いてください。

1. このフォルダでサーバを起動

```bash
python3 -m http.server 8000
```

2. ブラウザで開く

- http://localhost:8000

3. ブラウザのカメラ許可を「許可」

## あそびかた

1. `カメラON/OFF` を押してカメラをON
2. `ゲーム開始` を押す
   - 3→2→1→GO のカウントダウン後にスタート
3. 顔を左右・上下に動かして、緑のキャラを操作
4. 赤い玉に当たるとゲームオーバー

## カメラ選択

端末にカメラが複数ある場合、`カメラ選択` から使用するカメラを選べます。

## アイテム（5秒で消える）

ゲーム中にランダムで出現します。

- **C（青）**
  - 画面にいる敵をぜんぶ消します（フラッシュ演出あり）
- **S（黄）**
  - 拾ってしまうと **5秒間、敵の移動速度が2倍** になります（デバフ）

## ヘルプ

画面の `ヘルプ` ボタンを押すと、操作方法とアイテム説明をモーダルで確認できます。
